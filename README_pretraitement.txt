le script pretraitement_nettoiement_final.py

Ce script a pour l'objectif de traiter et/ou nettoyer les phrases et/ou les caractères selon votre besoin et également en même temps séparer le corpus en train et en test selon la patition que vous saissez. 

	entrée : un fichier csv
	sortie : deux fichiers, un fichier de train et un fichier de test avec pretraitement et nettoiement

Vous pouvez changer le nom du fichier d'entrée dans la lgine 82 du script : 
	file_in = "tweet_global_warming.csv"
